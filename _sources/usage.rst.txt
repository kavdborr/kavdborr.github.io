=====
Usage
=====

This project contains two python files: The main.py serves as the sccript executing a code example, while TurboTools.py includes the code to read, convert and plot turbomachinery maps.

.. toctree::
   :maxdepth: 1

   usage/main
   usage/TurboTools


4 folders are present, or will be generated by the code. The maps and convertedMaps folders contain the Gasturb input maps and TURBO output maps respectively.
The experiment folder contains experiment reports from EcosimPro simulations, which can be plotted on top of the maps.
The final figures folder contains saved png images of the plots.
